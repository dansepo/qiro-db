## ⚖️ QIRO 관리비 부과 방식 상세 설명

다음은 QIRO 시스템에서 관리비 항목에 설정할 수 있는 주요 부과 방식들입니다.

| No. | 부과 방식 명칭 (한글)             | 코드명 (영문 ID 예시)     | 설명                                                                                                | "관리비 항목 설정" 시 주요 설정값                                  | "월별 데이터 입력" 시 필요 정보 (해당 시)                                  | 관리비 산정 시 계산 예시 (세대별)                                                                 |
| :-: | :-------------------------------- | :------------------------ | :-------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------- | :-------------------------------------------------------------------------- | :---------------------------------------------------------------------------------------------- |
| 1   | 세대별 고정액 부과 | FIXED_AMOUNT_PER_UNIT   | 각 세대에 매월 동일한 고정 금액을 부과합니다.                                                         | 항목명, 고정 금액, 단위(원/세대), 과세여부, 적용기간         | 없음 (항목 설정 시 금액 고정)                                               | 설정된 고정 금액                                                                               |
| 2   | 면적 비례 부과 (단가 기반) | PER_AREA_RATE           | 각 세대의 특정 면적(예: 전용면적)에 비례하여, 설정된 면적당 단가를 곱해 부과합니다.                     | 항목명, 면적당 단가, 단위(원/㎡), 기준 면적 유형(전용/계약), 과세여부, 적용기간 | 없음 (단가와 호실 면적으로 자동 계산)                                       | 면적당 단가 × 해당 세대 기준 면적                                                              |
| 3   | 총액 면적 비례 배분 | TOTAL_COMMON_PER_AREA   | 건물 전체 또는 특정 공용 부분에 발생한 해당 항목의 '월별 총 발생액'을 각 세대의 면적에 비례하여 배분합니다. | 항목명, 배분 기준 면적 유형(전용/계약), 과세여부, 적용기간       | 해당 항목의 월별 총 발생액 (예: "월별 공용 관리비 입력" 화면에서 입력)       | (해당 세대 기준 면적 / 전체 배분 대상 세대 기준 면적 합계) × 월별 총 발생액                               |
| 4   | 총액 세대별 균등 배분 (N등분) | TOTAL_COMMON_PER_UNIT_EQUAL | 건물 전체 또는 특정 공용 부분에 발생한 해당 항목의 '월별 총 발생액'을 배분 대상 세대 수로 동일하게 나누어 부과합니다. | 항목명, 과세여부, 적용기간                                       | 해당 항목의 월별 총 발생액                                                | 월별 총 발생액 / 전체 배분 대상 세대 수                                                            |
| 5   | 총액 지분 비율 배분 | TOTAL_COMMON_PER_SHARE_RATIO | 건물 전체 또는 특정 공용 부분에 발생한 해당 항목의 '월별 총 발생액'을 각 세대가 가진 지분 비율에 따라 배분합니다. | 항목명, 과세여부, 적용기간. (각 세대별 지분 정보는 '호실 정보'에 사전 등록 필요) | 해당 항목의 월별 총 발생액                                                | (해당 세대 지분율 / 전체 지분율 합계 또는 100%) × 월별 총 발생액                                |
| 6   | (선택) 특정 사용량 비례 부과 | PER_CUSTOM_USAGE_RATE | 전기/수도/가스 외, 건물 내 특정 시설/서비스의 사용량에 따라 단가를 곱해 부과합니다. (예: 공용 세탁실 사용료) | 항목명, 사용량당 단가, 사용량 단위, 과세여부, 적용기간     | 각 세대별 해당 항목의 월별 사용량 (별도 사용량 입력 기능 필요)              | 사용량당 단가 × 해당 세대 월별 사용량                                                          |
| 7   | (선택) 인원수 비례 부과 | PER_OCCUPANT_RATE       | 세대에 거주(또는 등록된) 인원수에 비례하여 단가를 곱해 부과합니다. (잘 사용되지 않거나, 특정 커뮤니티 비용 등) | 항목명, 인당 단가, 단위(원/명), 과세여부, 적용기간. (각 세대별 인원수 정보 필요) | 없음 (단가와 호실 인원수로 자동 계산)                                       | 인당 단가 × 해당 세대 인원수                                                                   |
| 8   | (선택) 차량 대수 비례 부과 | PER_VEHICLE_RATE        | 세대에 등록된 차량 대수에 비례하여 단가를 곱해 부과합니다. (주차 관련 비용)                           | 항목명, 차량당 단가, 단위(원/대), 과세여부, 적용기간. (각 세대별 등록 차량 대수 정보 필요) | 없음 (단가와 세대 차량수로 자동 계산)                                       | 차량당 단가 × 해당 세대 등록 차량 대수                                                         |

------

**각 부과 방식에 대한 추가 설명:**

- **`세대별 고정액 부과 (FIXED_AMOUNT_PER_UNIT)`:**
  - 가장 단순한 방식으로, 모든 세대에 동일한 금액을 부과할 때 사용합니다. (예: 케이블TV료, 세대별 고정 청소비)
  - "관리비 항목 설정"에서 부과할 금액 자체를 입력합니다.
- **`면적 비례 부과 (단가 기반) (PER_AREA_RATE)`:**
  - 각 세대의 면적(전용 또는 계약)에 따라 관리비를 차등 부과할 때 사용합니다. (예: 일반관리비, 건물 유지보수 관련 비용 중 면적 기준 부과분)
  - "관리비 항목 설정"에서 면적당 단가를 입력합니다. 실제 세대별 금액은 (단가 × 세대 면적)으로 계산됩니다.
- **`총액 면적 비례 배분 (TOTAL_COMMON_PER_AREA)`:**
  - 매월 건물 전체에 대해 발생하는 특정 공용 비용(예: 공용 부분 보험료, 공용 복도 수선비 총액)이 있고, 이를 각 세대의 면적에 비례하여 나누어 부과할 때 사용합니다.
  - "관리비 항목 설정"에서는 이 항목명과 "면적 비례 배분"이라는 방식만 지정합니다.
  - 실제 매월 발생하는 **총액**은 "월별 공용 관리비 입력" 또는 "외부 고지서 금액 입력"과 같은 별도 기능을 통해 해당 청구월에 맞게 입력됩니다.
  - "관리비 산정" 시, 시스템은 입력된 총액을 가져와 각 세대의 면적 비율로 안분하여 부과합니다.
- **`총액 세대별 균등 배분 (N등분) (TOTAL_COMMON_PER_UNIT_EQUAL)`:**
  - 건물 전체에 발생하는 특정 공용 비용(예: 엘리베이터 정기 점검료 총액, 단지 미화 용역비 총액)을 모든 세대에 동일하게 나누어 부과할 때 사용합니다.
  - "관리비 항목 설정"에서는 항목명과 "세대별 균등 배분" 방식을 지정합니다.
  - 실제 매월 발생하는 **총액**은 별도로 입력받습니다.
  - "관리비 산정" 시, 시스템은 입력된 총액을 배분 대상 세대 수로 나누어 각 세대에 부과합니다.
- **`총액 지분 비율 배분 (TOTAL_COMMON_PER_SHARE_RATIO)`:**
  - (주로 아파트 등 집합건물) 건물 전체 발생 비용을 각 세대가 가진 소유 지분(등기부등본상 비율 등)에 따라 배분할 때 사용합니다.
  - "관리비 항목 설정"에서는 항목명과 "지분 비율 배분" 방식을 지정합니다. 각 세대의 지분 정보는 사전에 "호실 정보"에 입력되어 있어야 합니다.
  - 실제 매월 발생하는 **총액**은 별도로 입력받습니다.
- **선택적 부과 방식들 (`PER_CUSTOM_USAGE_RATE`, `PER_OCCUPANT_RATE`, `PER_VEHICLE_RATE`):**
  - 이러한 방식들은 건물의 특성이나 관리 규약에 따라 필요할 수 있는 추가적인 부과 기준입니다.
  - 각각 사용량, 인원수, 차량 대수 등 특정 기준값과 그에 대한 단가를 "관리비 항목 설정"에서 정의하거나, 관련 기준값을 별도로 입력받아 처리합니다.
  - **주의:** QIRO 서비스의 초기 버전에서는 가장 일반적으로 사용되는 1~5번 방식 위주로 지원하고, 6~8번과 같은 방식은 사용자 요구 및 시장 상황에 따라 추후 확장 기능으로 고려할 수 있습니다. 특히, 계량기 기반의 전기/수도/가스 사용량은 별도의 "검침 데이터 등록" 및 관련 요금 계산 로직을 통해 처리되므로, 여기서의 `PER_CUSTOM_USAGE_RATE`는 그 외의 사용량 기반 항목을 의미합니다.

**시스템 구현 시 고려사항:**

- "관리비 항목 설정" UI에서 `부과 방식`을 선택하면, 해당 방식에 필요한 입력 필드(예: 단가, 단위, 기준 면적 유형 등)만 동적으로 표시되거나 활성화되어야 사용자 혼란을 줄일 수 있습니다.
- "총액 배분" 방식의 항목들은, "월별 공용 관리비 입력" 기능 등에서 해당 월의 총액이 입력되지 않으면 "관리비 산정" 시 해당 항목은 '0'으로 처리되거나 "데이터 미입력"으로 표시되어야 합니다.

이처럼 다양한 부과 방식을 제공하고, 각 방식의 의미와 필요한 입력값을 명확히 안내함으로써 QIRO 사용자는 자신의 건물 특성에 맞는 정확하고 합리적인 관리비 부과 체계를 구성할 수 있을 것입니다. 😊
